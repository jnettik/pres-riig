The solutions

Disclaimer: This is not an in depth explaination of responsive image solutions.

There are much better resources, many of which are linked to in the resources slide.

Goal here is to roughly understand what we're trying to accomplish through Drupal's GUI.

img tag

This is required for every solution. Every other technique builds on to HTML's core img tag.

srcset attribute

Attribute to inform the browser of alternate image sources.

Best used when trying to address resolution use cases.

srcset attribute

srcset comes in two flavors:

X descriptors tell the browser this image is suited for specific screen resolutions. This is great for fixed width images. Drupal seems to always default to this.
W descriptors inform the browser of the width of the image sources, then use the sizes attr to provide information about how that image will be displayed.
using that info, browser can pick more appropriate sources.

picture element

New element in HTML5 to define different image sources based on varying criteria.

srcset allows the browser to make educated decisions on source.

picture dictates to the browser which source we want.

Best when used for art direction or browser support use cases.

picture element


Media condition: stripped down media queries.

srcset: defines the source to use at this condition.

srcset is fully featured. So you can then support resolution switching on your art directed images.

Complex problems, complex solutions

There is an easier way!

Drupal FTW!

Drupal has awesome modules to ease a lot of this complexity.

Image Styles: Create all the different image sizes and crops for us.

Breakpoints: Allows you to expose site breakpoints to Drupal, and group them by components.

Picture: Maps image styles to breakpoints. Creates image field formatters for both srcset with sizes and picture element. And it includes picturefill 2 polyfill for older browser support.

Image styles: using style guide can pre plan these styles, document them and even import them into drupal to save you a lot of time.
